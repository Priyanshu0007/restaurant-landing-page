## Run Locally

Clone the project

```bash
  git clone https://github.com/Priyanshu0007/resturent-landing-page.git
```

Go to the project directory

```bash
  cd restaurant-landing-page
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
